Sobre o SensorView

O que é?

O SensorView é um sistema web desenvolvido com Django para receber, armazenar e exibir leituras de temperatura e umidade de uma residência. Os dados são coletados por um sensor físico (DHT11 ou DHT22) conectado a um microcontrolador (ESP32 ou Arduino), que os envia ao sistema via HTTP.

Funcionalidades
  • Exibição da leitura atual de temperatura e umidade
  • Histórico de leituras por data e hora
  • Gráfico de variação ao longo do dia
  • Endpoint para receber dados do sensor via HTTP POST
Tecnologias utilizadas
Django
Bootstrap 5
Chart.js
ESP32 / Arduino
DHT11 / DHT22
